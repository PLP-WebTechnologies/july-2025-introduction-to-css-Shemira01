Understanding CSS Basics

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Key CSS Concepts

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. A CSS rule contains two parts: a selector and a declaration.

Selectors

Selectors are patterns used to select the element(s) you want to style. They can be based on element type, class, ID, attributes, or relationships between elements.

Properties

CSS properties determine the styles applied to selected elements. Examples include color, font-size, margin, padding, and many more.

Values

Each property is assigned a value. For example, a color property might have a value of "red" or "#FF0000", and a margin property might have a value of "20px".

The CSS Box Model

The CSS box model is a fundamental concept that describes the rectangular boxes generated for elements in the document tree. Each box has four areas: content, padding, border, and margin.

Content Area

This area contains the actual content of the element, such as text, an image, or other media. Its dimensions are the content width and content height.

Margin Area

The margin is the outermost layer that surrounds the box. It creates transparent space outside the border, separating the element from other elements.

Border Area

The border extends around the padding (if any) and content. Borders can be styled with different colors, widths, and styles.

Padding Area

The padding extends around the content area. It creates space inside the element, between the content and the border.

Content Area

This is the actual content of the element, where text and images appear.
